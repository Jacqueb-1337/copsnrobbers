# QUICK START - Copy & Paste Commands

## Step 1: Get Your PC IP

```powershell
ipconfig | findstr "IPv4"
```

Look for something like: `192.168.1.100` or `192.168.18.100`

---

## Step 2: Download Photon Server

1. Go to: https://www.photonengine.com/en-US/OnPremise
2. Download SDK zip file
3. Extract to: `d:\Projects\copsnrobbers\photon_setup\PhotonServer\`

---

## Step 3: Get Free License

1. Go to: https://dashboard.photonengine.com/
2. Sign up (free)
3. Get "Indie License" (free)
4. Download `.license` file
5. Copy to: `d:\Projects\copsnrobbers\photon_setup\PhotonServer\bin_Win64\`

---

## Step 4: Patch APK

Replace `192.168.1.100` with YOUR IP:

```powershell
cd d:\Projects\copsnrobbers
python patch_apk_for_local_server.py target_compiled_final.apk --server-ip 192.168.1.100 --output cnr_patched
```

---

## Step 5: Install on Phone

```powershell
adb uninstall com.joydo.minestrikenew
Start-Sleep -Seconds 2
adb install cnr_patched.apk
```

---

## Step 6: Start Photon Server

**Option 1 - Double-click:**
```
d:\Projects\copsnrobbers\run_photon_server.bat
```

**Option 2 - PowerShell:**
```powershell
& 'd:\Projects\copsnrobbers\photon_setup\PhotonServer\bin_Win64\PhotonControl.exe'
```

---

## Step 7: Test

1. On phone: Connect to same WiFi as PC
2. Launch Cops n Robbers
3. Go to Multiplayer → Worldwide
4. Should connect instantly!

---

## If Phone Can't Connect

Open PowerShell as Administrator:

```powershell
New-NetFirewallRule -DisplayName "Photon" `
  -Direction Inbound -Action Allow -Protocol UDP -LocalPort 5055
```

Then retry on phone.

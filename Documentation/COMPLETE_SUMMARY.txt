## 🎮 How It Works

### Architecture
```
┌─────────────────────┐
│     Your PC         │
│ ┌─────────────────┐ │
│ │ Server.exe      │ │ Listens on UDP 5055
│ │ • Rooms         │ │
│ │ • Players       │ │
│ │ • State Sync    │ │
│ └─────────────────┘ │
└─────────────────────┘
        ↕ UDP
┌──────┬──────┬──────┐
│Phone1│Phone2│Phone3│ Connect to 127.0.0.1:5055
└──────┴──────┴──────┘ (or your PC's IP)
```

### Game Flow
```
1. Server starts listening on port 5055
2. Player 1 launches patched APK
3. Game connects to server
4. Player 1 creates room "Game1"
5. Player 2 launches game, joins "Game1"
6. Both in same match - multiplayer works!
7. Server syncs player positions every 100ms
8. RPC broadcasts: kills, scores, events
```

### Data Sync
```
Player 1 Update (100ms)  ──┐
Player 2 Update (100ms)  ──┤─ Server ─┬─→ Player 1
Player 3 Update (100ms)  ──┤          └─→ Player 2,3
Event (Kill, etc.)       ──┘
```

## 📋 Step-by-Step Quick Start

### 1. Compile Server (1 minute)
```powershell
cd d:\Projects\copsnrobbers
csc CopsNRobbersServer.cs -out:CopsNRobbersServer.exe
```

### 2. Start Server (1 minute)
```powershell
.\CopsNRobbersServer.exe

# You should see:
# [✓] Server started on 0.0.0.0:5055
# [✓] Listening for connections...
```

### 3. Patch APK (2 minutes)
```powershell
# Get your PC's IP (or use 127.0.0.1 for local testing)
ipconfig | findstr IPv4

# Patch the APK
python apk_patch_server.py target_compiled_final.apk game_local.apk 127.0.0.1 5055
```

### 4. Install on Phone (2 minutes)
```powershell
adb install game_local.apk
```

### 5. Test (5 minutes)
- Launch game on phone
- Select "Multiplayer"
- Create a room
- Check server console for connection message ✓
- Success! 🎉
